# **App Name**: VIT GambleVerse

## Core Features:

- Balance Display: Displays the user's current balance.
- Deposit Funds: Allows users to deposit funds into their account.
- Simulate Gamble: Simulates the wheel and determines win/loss based on matching numbers.
- Calculate Payout: Calculates the payout based on the winning number and the bidding amount.
- Random Number Generation: Generates random numbers using JavaScript to determine outcomes.
- Bidding System: Implements a bidding system allowing the user to select how much they bid per round.

## Style Guidelines:

- Primary color: Cyan (#00FFFF) to convey a high-tech, futuristic feel.
- Background color: Very dark grey (#121212) to provide contrast and a modern feel.
- Accent color: Electric blue (#7DF9FF) for highlights and interactive elements.
- Headline font: 'Orbitron', sans-serif, for a futuristic, digital look. Note: currently only Google Fonts are supported.
- Body font: 'PT Sans', sans-serif, to provide excellent readability in longer runs of text. Note: currently only Google Fonts are supported.
- Use simple, line-based icons to keep the interface clean.
- Modern, full-screen layout with clear sections for balance, deposit, and game areas.
- Use subtle animations for the number wheel spinning and win/loss notifications.
